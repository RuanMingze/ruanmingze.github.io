## 本地预览

1. 克隆仓库：

```bash
git clone https://github.com/ruanmingze/ruanmingze.github.io.git
cd ruanmingze.github.io
```

2. 使用任意静态服务器预览（例如 Python）：

```bash
python3 -m http.server 8000
# 然后访问 http://localhost:8000
```
